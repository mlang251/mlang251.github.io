Tab-navigable accessibility

View the code for keyboard-navigability on my Github

I wanted to learn how to make my website more accessible to users with disabilities, so I researched accessibility in the W3C specification for Accessible Rich Internet Applications, and outfitted this website with ARIA roles, properties, and labels. I also ensured that this website's navigation links were fully accessible from the keyboard, using best practices as outlined in the article, Keyboard-navigable Javascript Widgets, published by the Mozilla Development Network. The keyboard-navigability was done using object-oriented Javascript, and had a huge impact on the way people with disabilities can browse my website. Using the tab, arrow, and enter/space keys ensures that people with visual or motor impairements can navigate my webite easily and fluidly.

Batch script for initializing ReactJS projects

View my ReactInit repository on Github

This project fulfilled two goals of mine: To create a simple way to initialize React projects, and to learn about batch scripting in Windows. To do this, I followed the article, React Setup, published by Codecademy.com, to get a step by step guide on what to download, which commands to run, and which files need to be created in order to initalize a React project. I then learned about batch scripting and created a .bat file which automates the entire process. Initializing React projects is now as easy as typing in a single script call to the command line

Fun fact: I created this batch script before I knew Create React App existed.
