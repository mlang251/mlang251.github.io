Tab-navigable accessibility

Purpose To provide a fully accessible site that is simple and easy to navigate

I wanted to learn how to make my website more accessible to users with disabilities, so I researched accessibility in the W3C specification for Accessible Rich Internet Applications, and outfitted my website with ARIA roles, properties, and labels. I also ensured that my website was fully navigable using the keyboard, as outlined in the article, Keyboard-navigable Javascript Widgets, published by the Mozilla Development Network. The keyboard-navigability was done using object-oriented Javascript, and had a huge impact on the way people with disabilities can browse my website. Using the tab, arrow, and enter/space keys ensures that people with visual or motor impairements can navigate my webite easily and fluidly.

View the code for keyboard-navigability on my Github

Batch script for initializing ReactJS projects

Purpose To create a simple way to run the CLI for the beginning steps of making a project in React

This project fulfilled two goals of mine: To create a simple way to initialize React projects, and to learn about batch scripting in Windows. To do this, I followed the article, React Setup, published by Codecademy.com, to get a step by step guide on what to download, which commands to run, and which files need to be created in order to initalize a React project. I then learned about batch scripting and created a .bat file which automates the entire process. Initializing React projects is now as easy as typing in a single script call to the command line
